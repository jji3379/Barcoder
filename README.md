### 담당 역할
* 바코드 스캔 API
* AWS 환경 구축 (Elastic Beanstalk, RDS, S3, CloudFront, Route53, ACM, SMS)
* Github Action CI/CD 환경 구축
* 문자 인증 API
* Spring Boot 에러 로그 Slack 연동
